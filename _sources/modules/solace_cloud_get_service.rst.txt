solace_cloud_get_service -- Get the Solace Cloud Service details.
=================================================================

Get the Solace Cloud Service details by name or serviceId.




Examples
--------

.. code-block:: yaml+jinja

   
     hosts: all
     gather_facts: no
     any_errors_fatal: true
     collections:
       - solace.pubsub_plus
     tasks:
       - name: "Get service details"
         solace_cloud_get_service:
           api_token: "{{ api_token_all_permissions }}"
           service_id: "{{ sc_service_created_id }}"
         register: get_service_result

       - set_fact:
           sc_service_created_info: "{{ result.response }}"

       - name: "Save Solace Cloud Service Facts to File"
         copy:
           content: "{{ sc_service_created_info | to_nice_json }}"
           dest: "./tmp/facts.solace_cloud_service.{{ sc_service.name }}.json"
         delegate_to: localhost

Parameters
----------

api_token (required)
  The API Token.

  Generate using Solace Cloud console with the appropriate permissions for the operations you want to enable.

  | **type**: str

name (optional)
  The name of the service.

  Note: If name is not provided, service_id must.

  | **type**: str

service_id (optional)
  The service id.

  Note: If service_id is not provided, name must.

  | **type**: str

timeout (optional)
  Connection timeout in seconds for the http/s request.

  | **type**: int
  | **default**: 60





Return Values
-------------

rc
  return code, either 0 (ok), 1 (not ok)

  | **returned**: always
  | **type**: int
  | **sample**:

  .. code-block:: json

     {
     "rc": 0
   }

msg
  error message if not ok

  | **returned**: error
  | **type**: str
  | **sample**:

  .. code-block:: json

     "{}"

response
  The response from the get call. Differs depending on state of service.

  | **returned**: success
  | **type**: complex
  | **sample**:

  .. code-block:: json

     "{}"

  serviceId
    The service Id of the created service

    | **returned**: if service exists
    | **type**: str
    | **sample**:

    .. code-block:: json

     "{}"

  adminState
    The state of the service

    | **returned**: if service exists
    | **type**: str
    | **sample**:

    .. code-block:: json

     "{}"
